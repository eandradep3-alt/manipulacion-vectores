# Manipulación de Vectores

## Descripción

Este proyecto fue desarrollado para la asignatura de Técnica de Programación y tiene como objetivo aplicar conceptos de vectores, estructuras de control, algoritmos de búsqueda, cálculos estadísticos y métodos de ordenamiento utilizando JavaScript, HTML y programación orientada a eventos.

La aplicación permite generar vectores numéricos de diferentes dimensiones, visualizar sus elementos y realizar diversas operaciones estadísticas y matemáticas sobre los datos almacenados.

## Funcionalidades implementadas

### Gestión del vector

- Cargar vector con números aleatorios.
- Presentar los elementos almacenados.
- Vaciar el vector.

### Operaciones matemáticas

- Calcular valor mínimo.
- Calcular valor máximo.
- Calcular suma total.
- Calcular producto total.

### Medidas estadísticas

- Calcular media aritmética.
- Calcular mediana.
- Calcular moda.

### Algoritmos de ordenamiento

- Ordenamiento por Selección.
- Ordenamiento por Burbuja.
- Orden ascendente.
- Orden descendente.

## Funcionamiento general

1. Seleccionar la dimensión del vector.
2. Presionar "Cargar Vector" para generar números aleatorios.
3. Utilizar "Presentar Vector" para visualizar los datos.
4. Ejecutar cualquiera de las operaciones disponibles:
   - Valor mínimo
   - Valor máximo
   - Suma total
   - Producto total
   - Media
   - Mediana
   - Moda
5. Ordenar los elementos mediante los algoritmos de Selección o Burbuja.
6. Visualizar los resultados en el campo de respuesta.
